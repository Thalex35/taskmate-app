# TaskMate

Gestionnaire de devoirs et deadlines pour étudiants universitaires.

## Stack

- **Frontend** : React.js + React Router + Context API
- **Stockage** : Supabase

## Équipe

- Theodore Louisjuste
- Shadaie Lozier
- Rosar

## Fonctionnalités

- Inscription et connexion par compte personnel
- Dashboard avec statistiques et devoirs urgents
- Gestion complète des devoirs (ajout, modification, suppression)
- Filtres par matière, statut et priorité
- Gestion des matières avec couleurs personnalisées
- Mode sombre / Mode clair
